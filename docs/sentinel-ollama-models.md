# Sentinel + Ollama: Локальные AI-модели

## Рекомендуемые модели

### Для кодинга (без ограничений)
```bash
ollama run huihui-ai/Huihui-Qwen3.5-35B-A3B-abliterated
```
- **Qwen 3.5 35B (abliterated)** — без цензуры и отказов
- Идеально для: генерация любого кода, рефакторинг, system prompts
- RAM: ~20GB, GPU: опционально
- Не отправляет данные в облако

### Для быстрых задач
```bash
ollama run qwen2.5-coder:7b
```
- Лёгкая модель для автодополнения и быстрых правок
- RAM: ~5GB

### Для анализа кода
```bash
ollama run deepseek-coder-v2:16b
```
- Сильная модель для code review и архитектурных решений
- RAM: ~12GB

## Настройка Sentinel + Ollama

1. Установите Ollama: https://ollama.com/download
2. Скачайте модель: `ollama run huihui-ai/Huihui-Qwen3.5-35B-A3B-abliterated`
3. Sentinel Bridge автоматически определит локальный Ollama по адресу http://localhost:11434

## Преимущества локальных моделей

- **Приватность** — код не покидает вашу машину
- **Без rate limits** — генерируйте сколько нужно
- **Без цензуры** — модель не отказывает в выполнении задач
- **Без интернета** — работает полностью офлайн
- **Бесплатно** — никаких API-ключей и подписок
